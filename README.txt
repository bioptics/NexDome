For details on using the specific software packages provided, refer to the
text documents in each of the appropriate directories.

In this directory
Calibration.txt - A description of the dome calibration procedure and a
                  detailed description of the safety timeout mechanisms

In the arduino directory

Update.txt - The procedure used to update firmware on the controllers
Wiring.txt - A description of the arduino wiring

Ascom drivers for the NexDome are provided in the windows directory tree.
The instaler directory contains a pre-built ready to use ascom driver.
The ascom-sources directory contains full source code used to build the
installer package.


Plugins for SkyX have been written by Rodolphe Pineau.  They are available
for multiple platforms here:- https://rti-zone.org/macosx_x2dome_plugins.php

For Linux users, NexDome drivers are part of the base INDI installation,
and sources are available in the INDI repositories.
